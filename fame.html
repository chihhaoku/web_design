Best Project Award


2018 Spring Best Project

Members: Erik Czech, Daniel Finn, Zachary Schoff, Samuel Brace, Hassen Noureddine

Description

Gradebook Lite

It is generally difficult to use the gradebook feature in Blackboard system. This group of students was then developed a light version of gradebook system for each student to check their learning status efficiently. This project was selected as the best project by the instructor in Spring 2018.
