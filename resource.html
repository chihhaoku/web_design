Resources

Visualization Tools/Library/Framework

  • Tableau: An interactive visualization software. It can be combined with R. Free for students.
  • Power BI: A business analytics service provided by Microsoft.
  • D3.js: A JavaScript library to visualize data using HTML, SVG, and CSS.
    • See Scott Murray's D3 Tutorial.
  • Social Network Visualization: Gephi, Pajek, NetMiner, KeyLines, NodeXL

Web Development

  • W3Scoools: A quick reference for HTML, CSS, Boostrap, jQuery, JavaScript, and AngularJS.
  • Bootstrap: An open source toolkit for developing HTML, CSS, and JavaScript.
  • jQuery: A JavaScript library for quick and interactive web development.
  • AngularJS: A JavaScript framework for quick and interactive web development.
  • Tutorial Resources: Code School, Code Academy, CodingGround, KeyLines, Lynda.com (Free for LTU students)

Teamwork / Version Control

  • Github: A web-based Git version control repository hosting service
