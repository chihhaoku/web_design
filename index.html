This course provides both ‘theory’, e.g., web design and e-commerce and ‘practice’, e.g., HyperText Markup Language (HTML), Cascading Style Sheets (CSS), and JavaScript to train students to create web pages. Students will develop web pages with current technologies and practices. In addition, hands-on practice, website case projects, and web research are emphasized in this class.

Topics: HTML basics, CSS and CSS3, visual elements and graphics, web design principles, page layout, mobile design, tables and forms, multimedia, e-commerce, web promotion, JavaScript and jQuery.


Project

Develop a functional website based on the instructions. The grading rubrics include the structure and design of pages, accessibility and usability, group presentation, and techniques used in the project.
